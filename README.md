# parallax.js

```
<div class="wrapper">
    <div class="section section1"></div>
    <div class="section section2"></div>
    <div class="section section3"></div>
    <div class="section section4"></div>
    <div class="section section5"></div>
</div>

<script src="js/vendor/jquery.min.js"></script>

<script>
    $('.wrapper').parallax();
</script>
```
